== Description ==

Animated search form with Pure CSS3, replace search form with custom CSS styles.

== Installation ==

Unzip plugin files and upload them under your '/wp-content/plugins/' directory.

Resulted names will be: './wp-content/plugins/search-box/*'

Activate plugin at "Plugins" administration page.
